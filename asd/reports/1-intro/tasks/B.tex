\subsection{Зоопарк Глеба}

\subsubsection{Описание решения}

По формулировке задача напоминает проверку модифицированной скобочной последовательности на корректность. Представим, что разные буквы обозначают разные виды скобок \textit{(круглая, квадратная, фигурная и др.)}, а животные с ловушками --- тип скобки \textit{(открытая/закрытая)}. Только в задаче неважно, стоит ли сначала открытая скобка и затем закрытая или наоборот --- оба варианта подойдут.

Чтобы проверить корректность последовательности, введём стек, который хранит структуры с метаданными элементов: его тип, вид и относительный индекс среди данного вида.

Запустим цикл по всему массиву и будем добавлять элементы в стек. Если текущий элемент одного вида с вершиной стека, но разного типа --- удаляем вершину стека и переходим к следующему элементу.

Последовательность будет правильной, если стек после обхода массива будет пустой: значит, все животные попались в свои ловушки.

\subsubsection{Асимптотическая сложность}

\textit{Временная сложность решения} --- линейная \( \mathcal{O}(N) \). В алгоритме используется обход по всему массиву, где для каждого элемента выполняется константное число операций.

\textit{Пространственная сложность решения} --- линейная \( \mathcal{O}(N) \). Для хранения \( N \) входных символов выделена строка в памяти, которую впоследствии обрабатывают при помощи стека и массива, каждый длины \( N \) в худшем случае. 

\subsubsection{Листинг кода}

\begin{lstlisting}
#include <iostream>
#include <stack>
#include <vector>

struct item {
  char value;
  bool isTrap;
  int idx;
};

int main() {
  std::string zoo;
  std::cin >> zoo;
  int N = zoo.length();
  std::stack<item> stack;
  std::vector<int> order;
  for (int i = 0; i < N; i++) {
    order.push_back(-1);
  }
  int animal_idx = 1, trap_idx = 1;
  for (int i = 0; i < N; i++) {
    bool isTrap = (char)std::tolower(zoo[i]) != zoo[i];
    int idx = isTrap ? trap_idx++ : animal_idx++;
    item curr = {(char)std::tolower(zoo[i]), isTrap, idx};
    if (stack.size() == 0 || stack.top().value != curr.value || stack.top().isTrap == curr.isTrap) {
      stack.push(curr);
      continue;
    } else {
      item top = stack.top();
      stack.pop();
      if (top.isTrap) {
        order[top.idx - 1] = curr.idx;
      } else {
        order[curr.idx - 1] = top.idx;
      }
    }
  }
  if (stack.size() > 0) {
    std::cout << "Impossible";
  } else {
    std::cout << "Possible" << std::endl;
    for (int i = 0; i < N; i++) {
      if (order[i] != -1) {
        std::cout << order[i] << " ";
      }
    }
  }
  return 0;
}
\end{lstlisting}
