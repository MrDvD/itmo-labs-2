\section{Список задач}

В рамках выполнения домашней работы \textit{по варианту №10} необходимо предоставить решения следующих задач:

\begin{enumerate}
  \item Вычислить площадь фигуры, ограниченной кривыми:
    \[
      (x^2+y^2)^2=a(x^3-3xy^2),\quad a> 0
    \]
  \item Вычислить объём тела, ограниченного данными поверхностями:
    \[
      z = \dfrac{x^2}{a^2}+\dfrac{y^2}{b^2},\quad z=c
    \]
  \item Найти момент инерции относительно координатных плоскостей тела, ограниченного поверхностями:
    \[
      \dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=2\dfrac{z}{c},\quad\dfrac{x}{a}+\dfrac{y}{b}=\dfrac{z}{c}\quad(a> 0,\ b> 0,\ c> 0) 
    \]
  \item Вычислить \textit{(тройным интегралом)} объёмы тел, ограниченных поверхностями:
    \[
      z=\sqrt{64-x^2-y^2},\quad x^2+y^2\leq 60,\quad z=1
    \]
\end{enumerate}
