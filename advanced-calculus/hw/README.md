# Домашняя работа

## Обзор

Домашняя работа — это самостоятельное выполнение учащимися заданий вне пары для повторения, закрепления и углубления знаний, а также применения их на практике.

## Цель и значение

Домашняя работа помогает формировать самостоятельность, развивать умения самоконтроля, планирования и применения знаний в новых ситуациях.
