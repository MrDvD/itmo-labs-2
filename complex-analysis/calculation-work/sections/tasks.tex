\section{Список заданий}

В рамках выполнения расчётной работы \textit{по варианту №15} необходимо предоставить решения следующих заданий:

\begin{enumerate}
  \item Изобразить на комплексной плоскости множество \( \mathcal{D} \):
  %
  \[ \mathcal{D}=\{z\colon\abs{z-3+2i}\leq 2,\ 0<\Re(iz)\leq 1\} \]

  \item Найти все значения функции в указанной точке:
  %
  \[ e^{e^i} \]

  \item Найти аналитическую функцию по известной её действительной части:
  %
  \[ u(x,y)=\sh(2x)\cos(2y)+x^2-y^2+4y-4 \]

  \item Вычислить интеграл по заданной кривой в указанном направлении:
  %
  \[ \int_C \Im z\diff z,\quad C\colon\abs{z-1}=1,\ \Im z\geq 0,\ z_0=2 \]

  \item Разложить функцию \( f(z) \) в ряд Тейлора в окрестности точки \( z_0 \) и указать область, в которой ряд представляет данную функцию:
  %
  \[ f(z)=z^2 e^z,\quad z_0=1 \]

  \item Разложить функцию \( f(z) \) в ряд Лорана в указанной области:
  %
  \[ f(z)=\dfrac{3}{z^2-7z+10},\quad 2<\abs{z}<5 \]

  \item Вычислить интеграл при помощи вычетов:
  %
  \[ \int_L z^3\cos\left(\dfrac{2i}{z}\right)\diff z,\quad L=\{z\colon\abs{z}=\sqrt{2}\} \]

  \item Вычислить несобственный интеграл:
  %
  \[ \int_{-\infty}^{+\infty}\dfrac{x\cos(ax)}{x^2+4x+8}\diff x,\quad a>0 \]
\end{enumerate}
