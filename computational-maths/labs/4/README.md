# Решение систем нелинейных уравнений методом простых итераций

Реализация алгоритма численного решения системы нелинейных уравнений по заданному начальному приближению. Для поиска корней используется метод простых итераций, обеспечивающий точность до 5-го знака после запятой.

## Алгоритм работы

1.  **Подготовка системы**:
    Исходная система уравнений вида $F(X) = 0$ приводится к эквивалентному виду, удобному для итераций:
    $$x_i = \phi_i(x_1, x_2, \dots, x_n)$$
2.  **Начальное приближение**:
    В качестве старта используются значения $x_0, y_0, \dots$, переданные во входных данных.
3.  **Итерационный процесс**:
    На каждом шаге вычисляются новые значения неизвестных на основе значений с предыдущего шага:
    $$x^{(k+1)} = \phi(x^{(k)})$$
    Процесс повторяется до тех пор, пока не будет достигнута заданная точность (5 знаков после запятой).
4.  **Условие сходимости**:
    Метод сходится, если в окрестности корня выполняется условие достаточной малости производных функций $\phi$.

## Формат ввода
Программа ожидает строку, содержащую номер системы, количество уравнений и начальные приближения:
`k n x0 y0 ...`

* **k**: Порядковый номер системы уравнений.
* **n**: Количество уравнений и количество неизвестных.
* **x0, y0, ...**: Начальные приближения для соответствующих неизвестных.

## Формат вывода
Программа выводит список значений, представляющих найденные корни системы:
`x1 x2 ... xn`

> **Примечание**: Все значения выводятся с точностью до 5-го знака после запятой. Порядок корней соответствует порядку неизвестных во входных данных.
