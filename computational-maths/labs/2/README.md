# Решение СЛАУ методом Гаусса

Реализация алгоритма решения системы линейных алгебраических уравнений вида $Ax = b$ методом Гаусса с последующим вычислением вектора невязок.

## Алгоритм работы

1. **Входные данные**: Принимается размерность системы $n$, коэффициенты матрицы $A$ и значения вектора свободных членов $b$.
2. **Прямой ход**:
    * Матрица приводится к верхнетреугольному виду с помощью элементарных преобразований строк.
    * На каждом шаге выбирается ведущий элемент. Если все элементы в текущем столбце ниже главной диагонали равны нулю, проверяется определенность системы.
3. **Анализ решения**:
    * Если в процессе приведения к ступенчатому виду обнаруживается строка, где все коэффициенты при неизвестных равны нулю, а свободный член — нет, или если ранг матрицы меньше $n$, переменная `isSolutionExists` принимает значение `false`.
    * Выводится сообщение об отсутствии единственного решения.
4. **Обратный ход**:
    * Если решение существует, последовательно вычисляются значения неизвестных $x_i$, начиная с $x_n$.
5. **Расчет невязок**:
    * Для каждого уравнения системы вычисляется невязка $r_i$ по формуле: 
      $$r_i = \sum_{j=1}^{n} a_{ij}x_j - b_i$$
    * Где $x_j$ — найденные значения неизвестных, а $a_{ij}$ и $b_i$ — исходные коэффициенты.

## Формат ввода
Программа ожидает строку, содержащую размерность и элементы расширенной матрицы:
`n a11 a12 ... a1n b1 a21 a22 ... a2n b2 ... an1 an2 ... ann bn`

* `n`: Количество уравнений (и неизвестных).
* `a_ij`: Коэффициенты при неизвестных.
* `b_i`: Свободные члены.

## Формат вывода
* Если система имеет единственное решение:
  `x1 x2 ... xn r1 r2 ... rn`
  *(Где x — корни, r — невязки)*.
* Если система не имеет решений или имеет бесконечное множество:
  `The system has no roots of equations or has an infinite set of them.`
