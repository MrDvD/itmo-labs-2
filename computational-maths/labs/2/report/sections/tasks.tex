\section{Постановка задач}

Дана система линейных алгебраических уравнений вида \( Ax=b \), где \( A \) --- матрица коэффициентов размерности \( n \times n \), а \( b \) --- вектор свободных членов.

В \textit{варианте №1} решение системы осуществляется \textit{методом Гаусса}, а оценка корректности --- через \textit{расчёт вектора невязок}.

То есть в ходе лабораторной работы будут выполнены следующие задачи:

\begin{enumerate}
    \item Разработка программного модуля для реализации алгоритма Гаусса.
    \item Реализация проверки системы на совместность и определенность.
    \item Написание алгоритма вычисления вектора невязок \( r \) для оценки погрешности полученного результата.
    \item Вычисление вектора решений \( x \) с вектором невязок \( r \) для данной СЛАУ.
\end{enumerate}
