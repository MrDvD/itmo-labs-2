\section{Постановка задач}

Дана система нелинейных уравнений вида $F(X) = 0$. По заданному начальному приближению $X^{(0)}$ необходимо найти решение системы с точностью до 5-го верного знака после запятой.

В \textit{варианте №2} решение осуществляется \textit{методом простых итераций}.

То есть в ходе лабораторной работы будут выполнены следующие задачи:

\begin{enumerate}
    \item Приведение исходной системы нелинейных уравнений к итерационному виду $X = \Phi(X)$, который подходит для применения метода.
    \item Разработка алгоритма, который реализует метод простых итераций для системы уравнений.
    \item Вычисление вектора корней системы по заданным начальным приближениям и выбранному номеру системы.
\end{enumerate}
