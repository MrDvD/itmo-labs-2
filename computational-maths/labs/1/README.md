# Интерполяция полиномом Лагранжа по узлам Чебышёва

Реализация алгоритма построения интерполяционного полинома Лагранжа, где узлы интерполяции определяются как корни полиномов Чебышёва для заданного интервала $[a, b]$.

## Алгоритм работы

1.  **Входные данные**: Принимаются значения $f$ (индекс функции), $a$ и $b$ (границы интервала), а также точка $x$.
2.  **Выбор узлов**: Узлы интерполяции $x_i$ рассчитываются по формуле корней полинома Чебышёва, масштабированных под отрезок $[a, b]$.
3.  **Итерационный процесс**:
    * Начинается построение полинома Лагранжа $L_n(x)$.
    * Количество узлов $n$ последовательно увеличивается.
    * Вычисляется модуль разности значений функции в точке $x$ между текущей и предыдущей итерациями (или разность с самой функцией, если она известна).
    * Процесс останавливается, когда $|L_n(x) - L_{n-1}(x)| < 0.01$.
4.  **Выходные данные**: Значение интерполяционного полинома в заданной точке $x$.

## Формат ввода
Программа ожидает строку формата:
`f a b x`

* `f`: Номер функции.
* `a`: Левая граница интервала.
* `b`: Правая граница интервала.
* `x`: Точка, в которой ищется значение.

## Формат вывода
* Вещественное число (результат интерполяции).
