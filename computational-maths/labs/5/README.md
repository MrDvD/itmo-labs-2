# Решение задачи Коши для ОДУ методом Эйлера

Реализация алгоритма численного интегрирования обыкновенного дифференциального уравнения первого порядка вида $y' = f(x, y)$ на заданном интервале $[a, b]$. Для поиска приближённого решения применяется метод Эйлера с автоматическим уточнением шага интегрирования для обеспечения заданной точности $\varepsilon$.

## Алгоритм работы

1.  **Подготовка данных и выбор функции**:
    Исходные параметры считываются из входного потока. Правая часть уравнения $f(x, y)$ выбирается по указанному индексу с помощью метода `get_function`. Задаются границы интервала $[a, b]$, начальное условие $y(a) = y_0$ и параметр точности $\varepsilon$.
2.  **Инициализация и расчёт шага**:
    Задаётся начальный шаг интегрирования $h$. Шаг будет автоматически пересчитываться в процессе работы алгоритма (как правило, методом последовательного деления пополам) до тех пор, пока не будет достигнуто требуемое приближение значения $y(b)$.
3.  **Итерационный процесс интегрирования**:
    На каждом этапе вычисляются последовательные приближения решения по формуле метода Эйлера:
    $$y_{i+1} = y_i + h \cdot f(x_i, y_i)$$
    Процесс повторяется до достижения правой границы $x = b$. После получения значения $y_h(b)$ шаг уменьшается вдвое ($h_{new} = h/2$), и интегрирование выполняется заново для получения уточнённого значения $y_{h/2}(b)$.
4.  **Контроль точности и завершение**:
    Алгоритм проверяет условие сходимости $|y_{h/2}(b) - y_h(b)| \le \varepsilon$. При выполнении условия процесс останавливается, и в качестве результата возвращается значение $y_{h/2}(b)$. В противном случае шаг вновь уменьшается, и интегрирование повторяется. Дополнительно выполняется построение графиков численного и аналитического решений для анализа погрешности и устойчивости.

## Формат ввода
Программа ожидает строку, содержащую границы интервала, начальное условие, номер уравнения и параметр точности:
`a b y0 f_index epsilon`

* **a**: Левая граница интервала интегрирования.
* **b**: Правая граница интервала интегрирования.
* **y0**: Значение функции в начальной точке $y(a)$.
* **f_index**: Порядковый номер дифференциального уравнения для выбора функции $f(x, y)$ через метод `get_function`.
* **epsilon**: Параметр точности $\varepsilon$, определяющий максимальную допустимую разницу между последовательными итерациями.

## Формат вывода
Программа выводит приближённое значение решения в правой границе интервала:
`y(b)`

> **Примечание**: Значение выводится с точностью, соответствующей заданному параметру $\varepsilon$. В процессе работы также генерируются графики, отображающие численное решение, аналитическое решение (при его наличии) и начальную точку $(a, y_0)$, что позволяет визуально оценить накопление погрешности и устойчивость метода Эйлера на выбранном интервале. Порядок параметров ввода может быть адаптирован под требования конкретной системы тестирования.
