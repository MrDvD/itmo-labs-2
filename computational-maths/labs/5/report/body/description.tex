\subsection{Описание метода}

\subsubsection{Постановка задачи Коши для ОДУ}

\textbf{Обыкновенным дифференциальным уравнением} называется уравнение, которое связывает независимую переменную $x$, искомую функцию $y(x)$ и её производные различных порядков.

Если в уравнение входит только первая производная, оно называется ОДУ \textit{первого порядка}:
\[
y' = f(x, y).
\]

\textbf{Задачей Коши} \textit{(или начальной задачей)} для дифференциального уравнения первого порядка называется задача поиска функции $y(x)$, удовлетворяющей самому уравнению на интервале $[a, b]$, а также заданному \textit{начальному условию}:
\[
y(a) = y_0,
\]
где $y_0$ --- известное начальное значение в точке $x = a$. 

Численное решение задачи Коши заключается в построении сетки узлов $\{x_i\}_{i=0}^n \subset [a, b]$ с шагом $h_i = x_{i+1} - x_i$ и нахождении приближённых значений $y_i \approx y(x_i)$.

\subsubsection{Метод Эйлера}

Рассмотрим простейший одношаговый численный метод --- \textbf{метод Эйлера} \textit{(метод ломаных)}. Пусть на отрезке $[a, b]$ задан постоянный шаг интегрирования $h = \frac{b-a}{n}$, тогда узлы сетки определяются как $x_i = a + ih$.

Разложим истинное решение $y(x)$ в окрестности точки $x_i$ в ряд Тейлора:
\[
y(x_{i+1}) = y(x_i + h) = y(x_i) + h y'(x_i) + \frac{h^2}{2} y''(\xi_i), \quad \xi_i \in (x_i, x_{i+1}).
\]

Учитывая, что $y'(x_i) = f(x_i, y(x_i))$, и отбрасывая члены высших порядков малости, получаем рекуррентную \textbf{расчётную формулу метода Эйлера}:
\[
y_{i+1} = y_i + h f(x_i, y_i), \quad i = 0, 1, \dots, n-1.
\]

\textit{Локальная погрешность} метода (погрешность на одном шаге) имеет второй порядок малости по шагу: $O(h^2)$. Однако при интегрировании на всем отрезке $[a, b]$ ошибки суммируются, поэтому \textit{глобальная (полная) погрешность} метода Эйлера на конце отрезка составляет:
\[
E_n = |y(b) - y_n| = O(h).
\]
Таким образом, метод Эйлера обладает первым порядком точности.

\subsubsection{Обоснование автоматического выбора шага по условию точности}

Для контроля глобальной погрешности и автоматического подбора шага применяется \textit{правило Рунге} (метод повторного просчета). Поскольку метод Эйлера имеет первый порядок точности ($O(h)$), приближенное решение $y_h(b)$, полученное с шагом $h$, связано с точным решением $y(b)$ соотношением:
\[
y(b) = y_h(b) + C \cdot h + O(h^2),
\]
где $C$ --- некоторая константа, не зависящая от $h$.

Если мы уменьшим шаг интегрирования в два раза и найдем решение $y_{h/2}(b)$, то для него справедливо:
\[
y(b) = y_{h/2}(b) + C \cdot \frac{h}{2} + O(h^2).
\]

Вычитая из второго равенства первое, можно исключить неизвестную константу $C$:
\[
y_{h/2}(b) - y_h(b) \approx C \cdot \frac{h}{2}.
\]

Отсюда следует, что главная часть глобальной погрешности для более точного решения $y_{h/2}(b)$ численно равна модулю разности решений на двух сетках:
\[
E_{h/2} = |y(b) - y_{h/2}(b)| \approx |y_{h/2}(b) - y_h(b)|.
\]

Таким образом, для достижения заданной пользователем точности $\varepsilon$ алгоритм должен итерационно уменьшать шаг $h$ (обычно делением пополам) до тех пор, пока не выполнится критерий останова:
\[
|y_{h/2}(b) - y_h(b)| \le \varepsilon.
\]
Выполнение этого условия гарантирует, что полученное численное значение на конце отрезка отличается от истинного аналитического решения не более чем на величину $\varepsilon$.
